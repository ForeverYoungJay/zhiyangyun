# 部署与运维

## 本地启动
```bash
docker compose up --build -d
cd backend && alembic upgrade head && cd ..
docker compose run --rm seed
```

## 健康检查
- `GET /health`
- Swagger：`/docs`

## 常用运维动作
- 重建：`docker compose up --build -d`
- 查看日志：`docker compose logs -f api`
- 数据回归：`python3 scripts/api_regression.py`

## 风险提示
- 当前无 Prometheus/Grafana/Loki 编排
- 审计日志未形成中间件落库
