# 数据模型字典（核心）

## 用户与权限
- `users`：账号主体
- `roles` / `permissions` / `user_roles` / `role_permissions`：RBAC 元数据

## A1 业务
- `elders`：长者档案
- `buildings/floors/rooms/beds`：资产与床位
- `service_items/care_packages/care_package_items/elder_packages`：护理服务域
- `care_tasks`：护理任务执行闭环
- `billing_items/billing_invoices`：计费与发票
- `vital_sign_records/health_assessments`：健康数据（含异常级别、随访任务关联、评估闭环状态）

## A2 运营
- `shift_*`、`approval_*`、`notification_*`、`training_*`（分模块表）

## B 端
- `family_accounts/family_visits/family_care_records/family_surveys`：家属门户
- `dashboard_metrics`：运营指标

## 本轮新增使用关系
- 家属服务增购：`b2-family/services/order` -> `elder_packages`
- 家属总览：聚合 `elders + vital_sign_records + elder_packages`
